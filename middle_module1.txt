1.HTML CSS 웹폰트 넣는 법과 안티앨리어싱
1)폰트 패밀리 설정법
body {
    font-family : '글씨체', '글씨체2';
}
-버그없이 사용하려면 폰트의 영문명을 사용
-폰트명에 띄어쓰기가 있을 수 있으니 따옴표 안에 담는게 좋음
-폰트명을 콤마로 여러개 쓰면 제일 왼쪽부터 폰트적용.
-실패하면 뒤에 있는 폰트들을 차례로 적용
-웹사이트 이용자의 컴퓨터에 설치가 된 폰트들을 적용할 수 있음

2)사용자의 컴퓨터에 설치되지 않은 폰트를 사이트에 이용하려면
@font-face {
    font-family : '이쁜폰트';
    src : url(nanumsquare.ttf);
}
-CSS파일최상단에 @font-face 라는 명령어를 넣고,
그 안에 적용할 폰트의 경로와 이름을 적어주면 됨(폴더 내에 폰트 파일이 필요)

3)웹폰트용 woff파일
-웹 폰트용으로 나온 woff파일이 있음(ttf에 비해 용량이 1/3수준)
-한글폰트 ttf파일은 용량이 매우 크기 때문에 웬만하면 woff파일을 사용하는걸 추천

4)폰트를 빠르게 사용하기 위한 Google Fonts
-구글폰트라는 사이트를 이용하면 굳이 폰트파일을 구하지 않아도 됨(fonts.google.com)
HTML에 첨부하고 싶다면 <link>로 시작되는 부분을 복붙하면 됨
css에 첨부하고 싶다면 @import로 시작되는 부분을 CSS맨 위에 복붙하면 됨
구글이 호스팅해주는 폰트가 미리 정의된 CSS 파일을 가져다 쓰는 것이라 내 사이트의 트래픽을 절약할 수 있는 장점이 있음
크롬 브라우저를 이미 방문한 사이트는 캐싱해주기 때문에 많은 사람들이 이용할수록 더 빠르게 폰트를 사용가능

5)폰트 Anti-aliasing에 대해
앤티앨리어싱이란 픽셀의 각진 부분을 스무스하게 바꿔주는 것임
ex)
transform : rotate(0.04deg);
CSS파일에다가 이렇게 작성
transform 요소는 요소를 살짝 회전시키는 스타일임
